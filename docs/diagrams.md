# Mermaid Schemes

Этот файл содержит полный набор `mermaid`-схем для проекта. Его удобно использовать как:

- единое место для просмотра всех алгоритмов;
- источник схем для презентации;
- материал для печати или вставки в отчёт.

## 1. Большая сквозная схема проекта

```mermaid
flowchart TD
    A["Открытый датасет CESNET-TimeSeries24 sample"] --> B["Выбор масштаба 1 hour"]
    B --> C["Загрузка agg_1_hour и times_1_hour"]
    C --> D["Выбор 5 наиболее активных и полных рядов"]
    D --> E["Агрегация n_bytes, n_packets, n_flows"]
    E --> F["Формирование модели нагрузки канала"]
    F --> G["Построение календарных признаков"]
    G --> H["Построение лагов 1, 2, 3, 6, 12, 24, 48, 72"]
    H --> I["Построение rolling mean и rolling std"]
    I --> J["Сдвиг признаков назад для исключения leakage"]
    J --> K["Разделение выборки на train и test по времени"]
    K --> L["Обучение NaiveLastValue"]
    K --> M["Обучение SeasonalNaive24"]
    K --> N["Обучение OLSRegression"]
    L --> O["Сравнение моделей по MAE, RMSE, MAPE, R2"]
    M --> O
    N --> O
    O --> P["Выбор лучшей модели"]
    P --> Q["Прогноз нагрузки на тестовом интервале"]
    Q --> R["Baseline: фиксированная ёмкость канала"]
    Q --> S["Adaptive policy: ёмкость по прогнозу и резерву"]
    R --> T["Сравнение с фактической нагрузкой"]
    S --> T
    T --> U["Расчёт overload_rate"]
    T --> V["Расчёт served_traffic_ratio"]
    T --> W["Расчёт total_overload_bytes"]
    T --> X["Расчёт unused capacity"]
    U --> Y["Сравнение baseline и adaptive policy"]
    V --> Y
    W --> Y
    X --> Y
    Y --> Z["Вывод: прогноз как основа управления ресурсом канала"]
```

## 3. Детальная схема подготовки данных

```mermaid
flowchart TD
    A["Архив ip_addresses_sample.tar"] --> B["Чтение agg_1_hour"]
    C["Архив times.tar"] --> D["Чтение times_1_hour"]
    B --> E["Выбор наиболее активных рядов"]
    E --> F["Агрегация по id_time"]
    D --> G["Восстановление timestamp"]
    F --> H["Объединение с временной шкалой"]
    G --> H
    H --> I["Построение признаков hour, day_of_week, month"]
    I --> J["Добавление is_weekend и is_holiday"]
    J --> K["Построение sin/cos признаков"]
```
